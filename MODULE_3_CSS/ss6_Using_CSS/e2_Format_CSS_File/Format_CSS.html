Viet Nam (ten chinh thuc Cong hoa Xa hoi chu nghia Viet Nam ) la mot quoc gia namo phia dong o giua Lao va Campuchia, thuoc ban dao Dong Duong

Phia Bac Viet Nam giap voi Trung Quoc, phia tay Viet Nam giap voi Lao va Campuchia, phia tay nam giap vinh Thai Lan, phia dong va phia nam giap bien Dong va co hon 4000 hon da bai da ngam lon nho, gan va xa bo, co vung noi thuy, lanh hai, vung dac quyen kinh te va them luc dia duoc chinh phu Viet Nam xac dinh
